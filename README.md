## 使⽤說明

加入好友開啟聊天，若無顯示選單，可點選 "選單" 按鈕開啟。
按下選單 "附近" 可搜尋距離該位置小於3公里內，文化設施或公共藝術之照片、名稱、創作者、作品材質，最多顯示5筆資料。
